## Fixed

- **The installer works again.** The v0.21.0 Windows installer failed for anyone who already had Warbell installed: the window flashed "Preparing to install…" and vanished, with Windows silently rolling the install back (error 1603) — no game, no error message. The build pipeline had a second, stale installer script that stamped **every** release with the same internal version number, so Windows saw the new package as a copy of the one already on disk and refused to lay it down. There is now one installer script for every build, its version tracks the real game version, and installing over an old copy removes it cleanly first.
- The installer also shows its proper setup wizard again and creates the **Desktop and Start-menu shortcuts** — the broken v0.21.0 package created neither.

**If you tried v0.21.0:** just run this installer — it removes the stranded old copy for you. (If Windows still objects, uninstall "Warbell" from *Apps & features* first.)

## Everything from v0.21.0

In short: **Potyczka**, a brand-new top-down RTS skirmish mode with base building, a four-resource economy, army training and a live AI opponent; a fix for the startup crash on integrated-graphics laptops; a first-person polish pass; and tidier world collision.
